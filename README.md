### 실행 방법
```bash
# 가상환경 실행 후
pip install -r requirements.txt

# 의존성 설치 이후
python app.py
```

- 옷은 몸에 딱 붙지 않게, 그러나 루즈하지 않게(슬림~레귤러핏)를 유지해 세련된 실루엣을 만드세요.
